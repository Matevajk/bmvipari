# Coming Soon 2028

This project is a simple "Coming Soon" webpage for the year 2028. It features a message indicating that the site is under construction and will automatically redirect users to the English version of the site at [bmvipari.hu/en](https://bmvipari.hu/en). Users can also switch to the Hungarian version at [bmvipari.hu/hu](https://bmvipari.hu/hu).

## Project Structure

```
coming-soon-2028
├── src
│   ├── index.html        # Main HTML document
│   ├── css
│   │   └── styles.css    # CSS styles for the webpage
│   └── js
│       └── main.js       # JavaScript for redirection and language switching
├── .gitignore            # Files and directories to ignore by Git
└── README.md             # Project documentation
```

## Setup Instructions

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd coming-soon-2028
   ```

3. Open the `src/index.html` file in a web browser to view the "Coming Soon" message.

## Features

- Automatic redirection to the English version of the site.
- Option to switch to the Hungarian version.
- Simple and clean design.
